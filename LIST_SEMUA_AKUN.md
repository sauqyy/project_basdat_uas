## ğŸ”„ Cara Reset dan Membuat Ulang Akun

Jika ingin membuat ulang semua akun dengan data fresh:

```bash
php artisan migrate:fresh --force
php artisan db:seed --class=FreshDataSeeder --force
```

Atau gunakan script batch (Windows):
```bash
reset_and_seed.bat
```

## ğŸ” Query untuk Melihat Semua Akun di Database

```sql
-- Lihat semua akun
SELECT 
    id,
    name,
    email,
    role,
    nip,
    prodi
FROM users
ORDER BY 
    CASE role
        WHEN 'admin_super' THEN 1
        WHEN 'admin_prodi' THEN 2
        WHEN 'dosen' THEN 3
        ELSE 4
    END,
    prodi,
    nip;
```
