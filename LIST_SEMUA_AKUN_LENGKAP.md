## ğŸ“Š Query SQL untuk Melihat Akun di Database

Anda bisa melihat daftar akun pengguna langsung dari database dengan query SQL berikut:

```sql
-- Semua akun
SELECT id, name, email, role, nip, prodi 
FROM users 
ORDER BY role, prodi, name;

-- Hanya dosen
SELECT id, name, email, nip, prodi 
FROM users 
WHERE role = 'dosen'
ORDER BY prodi, name;

-- Hanya admin prodi
SELECT id, name, email, nip, prodi 
FROM users 
WHERE role = 'admin_prodi'
ORDER BY prodi, name;
```

## â­ Preferensi Dosen

Setiap dosen memiliki:
- âœ… **Preferensi Global**: Minimal 3 hari dan minimal 3 pilihan jam
- âœ… **Preferensi Spesifik**: Untuk setiap mata kuliah yang diampu

Untuk melihat preferensi dosen:
```sql
SELECT 
    u.name as dosen,
    u.nip,
    pd.preferensi_hari,
    pd.preferensi_jam,
    pd.prioritas,
    pd.catatan
FROM preferensi_dosens pd
JOIN users u ON pd.dosen_id = u.id
WHERE pd.mata_kuliah_id IS NULL  -- Preferensi global
ORDER BY u.name;
```
